# Background Images Directory

This directory contains all local background images organized by page.

## Structure

- `home/` - Home page backgrounds (slideshow mode, 6-second rotation)
- `gallery/` - Gallery page backgrounds (random mode)
- `essays/` - Essays page backgrounds (random mode)
- `about/` - About page backgrounds (random mode)
- `shop/` - Shop page backgrounds (random mode)
- `contact/` - Contact page backgrounds (random mode)

## Adding Images

1. Place images in the appropriate folder
2. Update `/src/data/media.manifest.json` with image metadata
3. Include: id, path, alt text, and title

## Image Requirements

- Format: JPG, PNG, or WebP
- Recommended size: 1920x1080 or larger
- Optimize images before adding to reduce file size
